# powerbi-business-performance-analytics-dashboard
Overview

This project presents an interactive Power BI dashboard designed to analyze overall business performance. It focuses on key metrics like revenue, units sold, and marketing spend to provide meaningful insights for better decision-making.

 Key Features
 Revenue Analysis – Track total revenue and compare across states
 Units Sold Insights – Monitor product performance
 Marketing Spend Analysis – Evaluate spending efficiency
 Marketing vs Revenue Correlation – Understand impact of campaigns
 Customer Segmentation – Analyze customer decision patterns
 ROI Tracking – Measure marketing return on investment
 Predicted Revenue – Forecast future trends
